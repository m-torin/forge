# Webapp Template

A minimal starter template that combines Next.js with Mantine UI components and
Tailwind CSS for rapid application development.

## Overview

- **Type**: Template/Starter Project
- **Purpose**: Full-featured template for Next.js with Mantine v8,
  authentication, and enterprise features
- **Architecture**: Modern React application with integrated design systems

## Features

- ⚡ **Next.js App Router** - Latest Next.js features with app directory
- 🎨 **Mantine v8** - Modern React components library with forms
- 🎯 **Tailwind CSS v4** - Utility-first CSS framework
- 🔐 **Better Auth** - Complete authentication system with organizations
- 🌐 **Internationalization** - Multi-language support with next-intl v4
- 📊 **Analytics** - Built-in analytics tracking
- 🚀 **Feature Flags** - Dynamic feature management
- 🔍 **SEO Optimization** - Search engine optimization utilities
- 📱 **Responsive Design** - Mobile-first responsive layouts
- ⚙️ **TypeScript** - Full TypeScript support
- 🎨 **Design System** - UIX system integration
- 📈 **Observability** - Monitoring and logging built-in

## Technology Stack

### Core Technologies

- **Next.js** - React framework with App Router
- **React** - Component-based UI library
- **TypeScript** - Type-safe development

### UI & Styling

- **Mantine Core** - Comprehensive component library

## Quick Start

### Installation

```bash
# Clone or copy the template
cp -r apps/next-app-mantine-tailwind-template-main my-new-app

# Navigate to the project
cd my-new-app

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

### Development

```bash
# Development server
pnpm dev # Starts on port 3900

# Production build
pnpm build # Creates optimized production build

# Start production server
pnpm start # Serves the built application
```

## Extending the Template

### Adding New Components

1. Create components in `src/components/`
2. Use Mantine components with Tailwind utilities
3. Leverage the integrated theme system

### Adding New Pages

1. Create pages in `src/app/` directory
2. Follow Next.js App Router patterns
3. Maintain consistent styling approach

### Customizing Styles

1. Modify `theme.ts` for Mantine customizations
2. Update `globals.css` for global styles

## Migration Guide

### From Create Next App

1. Install Mantine dependencies
2. Copy theme configuration
3. Update layout with providers
4. Configure Tailwind with Mantine preset

### From Existing Mantine Project

1. Install Tailwind CSS
2. Add tailwind-preset-mantine
3. Configure PostCSS
4. Update styles for integration

This template provides the perfect starting point for projects that want to
combine the power of Mantine's component library with the flexibility of
Tailwind CSS, all built on Next.js's robust foundation.
