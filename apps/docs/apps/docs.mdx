# Documentation App

The documentation app is built with [Mintlify](https://mintlify.com) and
provides comprehensive documentation for the entire monorepo.

## Features

- **Modern Documentation Platform** - Built with Mintlify for beautiful, fast
  documentation
- **Comprehensive Coverage** - Documents all packages, apps, and development
  workflows
- **Interactive Examples** - Code examples and API references
- **Search Integration** - Full-text search across all documentation
- **Mobile Responsive** - Works perfectly on all devices

## Structure

The documentation is organized into several sections:

- **Repository Overview** - Architecture and development guides
- **Package Documentation** - Detailed docs for each package
- **App Documentation** - Documentation for each application
- **AI Hints** - Development hints and patterns for AI-assisted development

## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build documentation
pnpm build
```

## Configuration

The documentation is configured through `mint.json` and uses MDX files for
content.

## Deployment

The documentation is automatically deployed when changes are pushed to the main
branch.
