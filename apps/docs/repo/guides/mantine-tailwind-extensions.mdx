## üìã **ModalProvider**

### **Purpose**

Provides global modal management with consistent styling and behavior across
your application.

### **Features**

- ‚úÖ **Global Modal State** - Manage modals from anywhere in your app
- ‚úÖ **Consistent Styling** - Tailwind-like design with smooth animations
- ‚úÖ **Accessibility** - Proper focus management and keyboard navigation
- ‚úÖ **Backdrop Blur** - Modern backdrop with blur effect
- ‚úÖ **Responsive Design** - Adapts to different screen sizes

### **Usage**

```typescript
import { ModalProvider, useModals } from '@mantine/core';

// Wrap your app
function App() {
  return (
    <ModalProvider>
      <YourAppContent />
    </ModalProvider>
  );
}

// Use in components
function MyComponent() {
  const modals = useModals();

  const openModal = () => {
    modals.openModal({
      title: 'Example Modal',
      children: (
        <div>
          <p>This is a modal opened via ModalProvider</p>
          <Button onClick={() => modals.closeModal()}>Close</Button>
        </div>
      ),
    });
  };

  return <Button onClick={openModal}>Open Modal</Button>;
}
```

### **Configuration Options**

```typescript
<ModalProvider
  radius="md"           // Border radius
  shadow="lg"           // Shadow depth
  padding="md"          // Content padding
  centered={true}       // Center modal
  closeOnClickOutside={true}  // Close on backdrop click
  closeOnEscape={true}        // Close on Escape key
  withCloseButton={true}      // Show close button
/>
```

## üîî **NotificationsProvider**

### **Purpose**

Provides a toast notification system for displaying temporary messages to users.

### **Features**

- ‚úÖ **Multiple Positions** - Top/bottom, left/right/center positioning
- ‚úÖ **Auto-dismiss** - Configurable auto-close timing
- ‚úÖ **Smooth Animations** - Slide-in/out animations
- ‚úÖ **Icon Support** - Custom icons for different notification types
- ‚úÖ **Stacking** - Multiple notifications stack properly

### **Usage**

```typescript
import { NotificationsProvider, useNotifications } from '@mantine/core';

// Wrap your app
function App() {
  return (
    <NotificationsProvider>
      <YourAppContent />
    </NotificationsProvider>
  );
}

// Use in components
function MyComponent() {
  const notifications = useNotifications();

  const showNotification = () => {
    notifications.show({
      title: 'Success!',
      message: 'Your action was completed successfully',
      color: 'green',
      icon: <CheckIcon size={16} />,
    });
  };

  return <Button onClick={showNotification}>Show Notification</Button>;
}
```

### **Configuration Options**

```typescript
<NotificationsProvider
  position="top-right"    // Position: top-left, top-right, top-center, bottom-left, bottom-right, bottom-center
  zIndex={1000}          // Z-index for notifications
  containerWidth={320}   // Max width of notification container
  autoClose={4000}       // Auto-close delay in milliseconds
/>
```

### **Notification Types**

```typescript
// Success notification
notifications.show({
  title: 'Success',
  message: 'Operation completed',
  color: 'green',
  icon: <CheckIcon size={16} />,
});

// Error notification
notifications.show({
  title: 'Error',
  message: 'Something went wrong',
  color: 'red',
  icon: <XIcon size={16} />,
});

// Info notification
notifications.show({
  title: 'Information',
  message: 'Here is some info',
  color: 'blue',
  icon: <InfoIcon size={16} />,
});

// Warning notification
notifications.show({
  title: 'Warning',
  message: 'Please be careful',
  color: 'yellow',
  icon: <AlertTriangleIcon size={16} />,
});
```

## üé† **Carousel**

### **Purpose**

Enhanced carousel component for displaying images, content, or any elements in a
sliding interface.

### **Features**

- ‚úÖ **Infinite Loop** - Seamless looping through items
- ‚úÖ **Touch Support** - Swipe gestures on mobile
- ‚úÖ **Keyboard Navigation** - Arrow key support
- ‚úÖ **Indicators** - Visual progress indicators
- ‚úÖ **Controls** - Previous/next buttons
- ‚úÖ **Responsive** - Adapts to container size

### **Usage**

```typescript
import { Carousel } from '@mantine/core';

function MyCarousel() {
  return (
    <Carousel
      withIndicators
      loop
      withControls
      height={300}
    >
      <Carousel.Slide>
        <img src="/image1.jpg" alt="Slide 1" />
      </Carousel.Slide>
      <Carousel.Slide>
        <img src="/image2.jpg" alt="Slide 2" />
      </Carousel.Slide>
      <Carousel.Slide>
        <img src="/image3.jpg" alt="Slide 3" />
      </Carousel.Slide>
    </Carousel>
  );
}
```

### **Configuration Options**

```typescript
<Carousel
  radius="md"              // Border radius
  withIndicators={true}    // Show progress indicators
  loop={true}              // Enable infinite loop
  withControls={true}      // Show navigation controls
  controlSize="lg"         // Control button size
  height={300}             // Carousel height
  slideSize="100%"         // Slide width
  slideGap="md"            // Gap between slides
  align="start"            // Slide alignment
  slidesToScroll={1}       // Number of slides to scroll
/>
```

## üöÄ **Best Practices**

### **Performance**

1. **Lazy Load Modals** - Only render modal content when needed
2. **Limit Notifications** - Don't show too many notifications at once
3. **Optimize Images** - Use optimized images in carousels
4. **Debounce Editor** - Debounce editor onChange events

### **Accessibility**

1. **Keyboard Navigation** - All extensions support keyboard navigation
2. **Screen Readers** - Proper ARIA labels and roles
3. **Focus Management** - Automatic focus management in modals
4. **Color Contrast** - High contrast ratios for readability

### **Mobile Experience**

1. **Touch Gestures** - Carousel supports swipe gestures
2. **Responsive Design** - All extensions are mobile-friendly
3. **Touch Targets** - Adequate touch target sizes
4. **Viewport Adaptation** - Modals adapt to mobile viewports
