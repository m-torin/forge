## Quick Start

<Steps>
  <Step title="Clone and Setup">
    ```bash
    git clone <repository-url>
    cd forge
    pnpm install
    ```
  </Step>

  <Step title="Environment Configuration">
    ```bash
    # Download secrets for local development
    pnpm doppler:pull:all

    # Or use local environment files
    cp .env.example .env.local
    ```

  </Step>

  <Step title="Database Setup">
    ```bash
    # Run database migrations
    pnpm migrate

    # Generate Prisma client
    pnpm --filter @repo/database generate
    ```

  </Step>

  <Step title="Start Development">
    ```bash
    # Start all applications
    pnpm dev

    # Or start specific application
    pnpm dev --filter=docs
    ```

  </Step>
</Steps>

      await tx.product.create({ data: { name: 'Test Product' } });

## Best Practices

<Warning>
  **Development Guidelines:** - Always use TypeScript with strict mode enabled -
  Write tests for all new features and bug fixes - Follow the established
  package layer architecture - Use Zod for runtime validation and type safety -
  Keep dependencies up to date with `pnpm bump-deps` - Run `pnpm lint` and `pnpm
  typecheck` before committing
</Warning>

### Recommended Workflow

1. **Start Development**
   - Pull latest changes and install dependencies
   - Set up environment variables and database
   - Run tests to ensure everything works

2. **Feature Development**
   - Create feature branch from `main`
   - Write tests first (TDD approach)
   - Implement feature with proper TypeScript types
   - Ensure all tests pass and code is properly formatted

3. **Quality Assurance**
   - Run full test suite with coverage
   - Check for circular dependencies
   - Verify bundle size impacts
   - Test in different environments

4. **Code Review**
   - Create pull request with clear description
   - Ensure CI/CD checks pass
   - Address review feedback
   - Merge when approved

The development environment provides powerful tools and practices to build
high-quality, maintainable applications efficiently across the entire Forge
platform ecosystem.
