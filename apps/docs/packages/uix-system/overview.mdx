# UIX System

A comprehensive UI component system providing production-ready components for
authentication, data display, search, animations, and rich text editing. Built
with Mantine, Tailwind CSS, and TipTap Editor.

## Overview

The UIX System package provides a complete suite of UI components organized into
specialized categories:

- **Mantine Components** - Authentication flows, data tables, stats cards
- **Tailwind v3/v4** - Advanced animations, mobile optimizations, themes
- **TipTap Editor** - Rich text editing with collaboration
- **Shared Utilities** - Internationalization and common utilities

<Info>
  This package follows a modular approach where you import only what you need,
  keeping bundle sizes optimized.
</Info>

## Quick Start

### Installation

The UIX System is already configured as a workspace package. Install peer
dependencies as needed:

```bash
# Required dependencies (if not already installed)
pnpm add @mantine/core @mantine/form @mantine/hooks react react-dom

# Optional dependencies
pnpm add @mantine/modals @mantine/notifications @tiptap/core @tiptap/react yjs
```

### Basic Usage

#### Mantine Auth Components

```typescript
import { SignInForm, AuthGuard } from "@repo/uix-system/mantine";

function AuthPage() {
  return (
    <AuthGuard fallback={<SignInForm />}>
      <Dashboard />
    </AuthGuard>
  );
}
```

#### Tailwind Animations

```typescript
import { AdvancedAnimations, TypingAnimation } from "@repo/uix-system/tailwind/v3";

function HomePage() {
  return (
    <div>
      <TypingAnimation text="Welcome to our platform" />
      <AdvancedAnimations.FadeInUp>
        <h1>Animated Content</h1>
      </AdvancedAnimations.FadeInUp>
    </div>
  );
}
```

### Tailwind Components

  AdvancedAnimations,

  ParallaxEffect,
  KeyboardShortcuts

## Environment Configuration

### Required Variables

```bash
# For search functionality (when implemented)
# NEXT_PUBLIC_SEARCH_API_KEY=your-search-key

# For collaborative editing (optional)
NEXT_PUBLIC_COLLABORATION_ENDPOINT=wss://your-collab-server.com
NEXT_PUBLIC_YJS_WEBSOCKET_URL=wss://your-yjs-server.com
```

### Environment Setup

Add these to your app's `env.ts` file:

```typescript
export const env = createEnv({
  client: {
    NEXT_PUBLIC_COLLABORATION_ENDPOINT: z.string().optional(),
    NEXT_PUBLIC_YJS_WEBSOCKET_URL: z.string().optional()
  }
  // ... other config
});
```

## Architecture

### Design Principles

- **Modular**: Import only what you need
- **Composable**: Components work together seamlessly
- **Consistent**: Unified design language across all components
- **Accessible**: WCAG 2.1 AA compliance built-in
- **Performant**: Optimized for production use

### Technology Stack

- **Mantine v7** - Core component library
- **Tailwind CSS v3/v4** - Utility-first styling
- **TipTap Editor** - Rich text editing
- **Yjs** - Real-time collaboration
- **React 18** - Component framework
- **TypeScript** - Type safety

### Bundle Optimization

The package is optimized for minimal bundle impact:

- **Tree-shaking** - Only imported components are bundled
- **Code splitting** - Large components support dynamic imports
- **Peer dependencies** - Core libraries are shared
- **CSS optimization** - Unused styles are purged

This UI system provides everything needed for building modern, accessible, and
performant web applications with consistent design patterns.
