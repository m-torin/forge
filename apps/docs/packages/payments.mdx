# Payments Package

Minimal **Stripe SDK wrapper** with graceful degradation and **AI agent
toolkit** integration for payment processing.

## Overview

The payments package provides a lightweight Stripe integration with:

- **Server-Only Implementation**: Secure server-side Stripe SDK wrapper
- **Graceful Degradation**: Works without configuration for development
- **AI Agent Toolkit**: Stripe AI SDK for automated payment operations
- **Type-Safe Configuration**: Environment validation with T3 Env
- **Lazy Initialization**: Stripe client initialized on first use
- **Mock Responses**: Development-friendly mock objects when unconfigured

## Installation

```bash
pnpm add @repo/payments
```


## Basic Usage

### Server-Side Stripe Operations

```typescript
import { stripe } from "@repo/payments/server";

// Customer management
const customer = await stripe.customers.create({
  email: "customer@example.com",
  name: "John Doe",
  metadata: {
    organizationId: "org_123"
  }
});

// Payment intent creation
const paymentIntent = await stripe.paymentIntents.create({
  amount: 2000, // $20.00
  currency: "usd",
  customer: customer.id,
  metadata: {
    orderId: "order_456"
  }
});

// Subscription management
const subscription = await stripe.subscriptions.create({
  customer: customer.id,
  items: [
    {
      price: "price_1234567890",
      quantity: 1
    }
  ]
});
```

### Next.js Server Actions

```typescript
import {
  createPaymentIntentUtil,
  createCustomerUtil,
  createSubscriptionUtil
} from "@repo/payments/server/next";

// Use server action utilities
("use server");

export async function createPayment(formData: FormData) {
  const amount = Number(formData.get("amount"));
  const email = formData.get("email") as string;

  // Create customer
  const customer = await createCustomerUtil({
    email,
    name: formData.get("name") as string
  });

  // Create payment intent
  const paymentIntent = await createPaymentIntentUtil({
    amount,
    customer: customer.id
  });

  return paymentIntent;
}
```

### Client-Side React Hooks

```typescript
"use client";
import { useStripePaymentIntent, useStripeCustomer } from "@repo/payments/client/next";

export function PaymentForm() {
  const { createPaymentIntent, loading, error, paymentIntent } = useStripePaymentIntent({
    onSuccess: (pi) => console.log('Payment intent created:', pi.id),
    onError: (err) => console.error('Payment error:', err),
  });

  const { createCustomer, customer } = useStripeCustomer();

  const handleSubmit = async (formData: FormData) => {
    // Create payment intent via API call
    await createPaymentIntent({
      amount: 2000,
      currency: 'usd',
      metadata: { orderId: 'order_123' },
    });
  };

  return (
    <form action={handleSubmit}>
      {/* Payment form */}
      {loading && <p>Processing...</p>}
      {error && <p>Error: {error}</p>}
      {paymentIntent && <p>Payment intent: {paymentIntent.id}</p>}
    </form>
  );
}
```

## Error Handling

```typescript
import Stripe from "stripe";
import { logError } from "@repo/observability/server/next";

try {
  const paymentIntent = await stripe.paymentIntents.create({
    amount: 2000,
    currency: "usd",
    customer: customerId
  });
} catch (error) {
  switch (error.type) {
    case "StripeCardError":
      logError("Card payment declined", {
        error: error.message,
        code: error.decline_code,
        brand: error.payment_method?.card?.brand
      });
      break;
    case "StripeInvalidRequestError":
      logError("Invalid payment request", {
        error: error.message,
        param: error.param
      });
      break;
    default:
      logError("Payment processing error", {
        error: error.message,
        type: error.type,
        code: error.code
      });
  }
  throw error;
}
```

## Security Best Practices

<Warning>
  Always use server-side Stripe operations. Never expose secret keys to the
  client.
</Warning>

- **Server-Only**: All Stripe operations happen server-side
- **Webhook Verification**: Always verify webhook signatures
- **Metadata Usage**: Store internal IDs in Stripe metadata
- **Amount Validation**: Validate amounts before processing
- **Idempotency**: Use idempotency keys for critical operations

```typescript
// Use idempotency for critical operations
const paymentIntent = await stripe.paymentIntents.create(
  {
    amount: 2000,
    currency: "usd",
    customer: customerId
  },
  {
    idempotencyKey: `payment_${orderId}_${timestamp}`

```
