# Forge Documentation

Welcome to the Forge monorepo documentation. Forge is a modern, production-ready monorepo template
for building scalable applications.

<Note>
  **Forge** provides a complete foundation for building Next.js applications with TypeScript,
  authentication, payments, and more.
</Note>

## Quick Start

```bash
# Clone the repository
git clone https://github.com/your-org/forge.git

# Install dependencies
pnpm install

# Run development server
pnpm dev
```

## What's Included

- **Apps**: Web app, admin panel, web template, workers
- **Packages**: Shared UI components, utilities, and configurations
- **Infrastructure**: Database, authentication, payments, analytics
- **Developer Experience**: TypeScript, ESLint, Prettier, testing
