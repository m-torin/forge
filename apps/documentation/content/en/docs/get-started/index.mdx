# 🚀 Get Started

Get up and running with Forge in minutes.

## Prerequisites

- **Node.js 22+** - Required for ESM support
- **pnpm 10.6.3+** - Package manager
- **PostgreSQL** - Database (local or hosted)

## Quick Setup

```bash
# Clone the repository
git clone https://github.com/your-org/forge
cd forge

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local

# Set up database
pnpm migrate

# Start development
pnpm dev
```

## Environment Variables

Create `.env.local` files in each app with required variables:

```bash
# Database
DATABASE_URL="postgresql://user:password@localhost:5432/forge"

# Authentication
BETTER_AUTH_SECRET="your-secret-key"
BETTER_AUTH_URL="http://localhost:3200"

# Analytics (optional)
NEXT_PUBLIC_POSTHOG_KEY="phc_your_key"

# Email (optional)
RESEND_API_KEY="re_your_key"
```

## Development Commands

```bash
pnpm dev                # Start all apps
pnpm dev --filter=web   # Start specific app
pnpm build              # Build all apps
pnpm test               # Run tests
pnpm lint               # Lint and format
```

## Project Structure

- `/apps/web` - Main application (Port 3200)
- `/apps/backstage` - Admin panel (Port 3300)
- `/apps/workers` - Background jobs (Port 3400)
- `/packages/*` - Shared packages
