# 🌐 Web Application

The main marketing website with user authentication, internationalization, and comprehensive feature
showcase.

import { Callout } from 'nextra/components';

<Callout type="info">
  Web runs on port **3200** and serves as the primary user-facing application with full e-commerce
  capabilities.
</Callout>

## Core Features

- **Complete Auth Flow** - Sign-in, sign-up, magic links, social providers
- **Internationalization** - Multi-language support with locale routing
- **E-commerce** - Product catalog, cart, checkout, and order management
- **User Dashboard** - Account settings, order history, favorites
- **Advanced Routing** - Parallel and intercepting routes
- **Search Integration** - Full-text search with filters
- **Analytics Tracking** - Comprehensive event tracking with feature flags
- **SEO Optimized** - Server-side rendering with metadata

## API Endpoints

### Protected Routes

All API routes require authentication except `/api/auth`:

```typescript
// Protected user data
GET /api/protected
{
  "user": {
    "id": "user-123",
    "email": "user@example.com",
    "locale": "en"
  }
}

// Secure signup with bot protection
POST /api/auth/secure-signup
{
  "email": "new@example.com",
  "password": "secure-password",
  "name": "New User"
}
```

### Rate Limiting

| Endpoint      | Limit        | Window   |
| ------------- | ------------ | -------- |
| Auth          | 10 requests  | 1 minute |
| Secure Signup | 3 requests   | 1 hour   |
| Protected     | 100 requests | 1 minute |

## Security Features

### Middleware Stack

```typescript
// middleware.ts
const middleware = async (request) => {
  // 1. Internationalization
  const i18nResponse = internationalizationMiddleware(request);

  // 2. Authentication check
  const authResponse = await authMiddleware()(request);

  // 3. Arcjet bot protection
  await secure(['CATEGORY:SEARCH_ENGINE', 'CATEGORY:PREVIEW', 'CATEGORY:MONITOR'], request);

  // 4. Security headers
  return securityHeaders();
};
```

### Bot Protection

Arcjet integration for advanced bot detection:

```typescript
// Secure signup endpoint
const decision = await aj.protect(request);

if (decision.isDenied()) {
  return NextResponse.json({ error: 'Bot detected' }, { status: 403 });
}
```

```typescript

analytics.capture('product_viewed', {
  productId: product.id,
```
