# Storybook

Component documentation and testing environment for the design system.

## Overview

Storybook provides an isolated environment for developing, testing, and documenting UI components
from the design system packages.

## Key Features

- Component playground and testing
- Interactive documentation
- Visual regression testing
- Accessibility testing
- Multiple viewports and themes

## Development

```bash
cd apps/storybook
pnpm dev
```

Runs on port 3700.

## Stories

Component stories are defined throughout the design system packages and aggregated in Storybook for
unified documentation.

## Usage

Essential for component development, testing, and maintaining consistency across the design system.
