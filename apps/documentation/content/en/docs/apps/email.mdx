# Email

Email template development and preview application using React Email.

## Overview

Provides a development environment for creating, testing, and previewing email templates with React
Email components.

## Key Features

- React Email integration
- Live preview of email templates
- Development server for testing
- Export to static HTML

## Development

```bash
cd apps/email
pnpm dev
```

Runs on port 3500.

## Templates

Email templates are located in `/emails` and built with React Email components for consistent
rendering across email clients.

## Usage

Used in conjunction with the `@repo/email` package for production email sending.
