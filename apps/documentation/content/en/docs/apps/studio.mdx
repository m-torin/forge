# Studio

Database administration and exploration tool using Prisma Studio.

## Overview

Provides a web-based interface for viewing, editing, and managing database records through Prisma
Studio.

## Key Features

- Visual database browser
- Record editing and creation
- Relationship visualization
- Query execution
- Data export capabilities

## Development

```bash
cd apps/studio
pnpm dev
```

Runs on port 3600.

## Usage

Essential tool for database development, debugging, and data management during development and
testing phases.

## Access

Connects to the same database as the main applications, providing a convenient GUI for database
operations.
