# Documentation

The main documentation site for the Forge monorepo, built with Nextra 4.

## Overview

This is the central documentation hub for all apps and packages in the Forge monorepo, providing
comprehensive guides, examples, and API references.

## Key Features

- Nextra 4 documentation framework
- MDX support with React components
- Full-text search with Pagefind
- Internationalization support
- Dark mode support

## Development

```bash
cd apps/documentation
pnpm dev
```

Runs on port 3800.

## Structure

- `/docs` - Main documentation content
- `/examples` - Code examples and demos
- Built-in search and navigation

## Adding Content

Create new `.mdx` files in the appropriate directories and update `_meta.tsx` files to include them
in navigation.
