# 🎨 Design System

Forge uses **Mantine UI v8** as the primary component library with custom composite components.

## Core Components

- **Authentication UI** - Unified sign-in/up forms with feature flag integration
- **Layout Components** - Responsive app shells and navigation
- **Form Components** - Enhanced form handling with validation
- **Data Display** - Tables, cards, and lists with analytics

## Usage

```typescript
import { UnifiedSignIn, AuthLayout } from '@repo/design-system/uix';

<AuthLayout title="Welcome Back">
  <UnifiedSignIn
    showSocialOptions={true}
    showMagicLink={true}
  />
</AuthLayout>
```

## Theme Configuration

Mantine v8 with dark mode support, custom color schemes, and responsive design patterns.
