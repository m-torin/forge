# Next.js Config Package

Shared Next.js configuration wrapper for consistent app setup.

## Overview

Provides a standardized Next.js configuration wrapper that includes common optimizations, security
headers, and build settings for all Next.js applications in the monorepo.

## Key Features

- Shared configuration wrapper
- Security headers included
- Build optimizations
- Environment-specific settings
- Consistent defaults

## Installation

```bash
pnpm add @repo/next-config
```

## Basic Usage

```typescript
import config from '@repo/next-config';

export default config({
  // Your custom Next.js config
  experimental: {
    serverActions: true,
  },
});
```

## Features

- **Security**: Built-in security headers
- **Performance**: Optimized build settings
- **Environment**: Development vs production configurations
- **Extensions**: Easy customization and extension

## Default Settings

Includes sensible defaults for:

- Bundle optimization
- Image optimization
- Security headers
- TypeScript configuration

## Customization

The wrapper accepts standard Next.js configuration options and merges them with the shared defaults.
