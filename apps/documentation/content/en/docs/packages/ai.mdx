# AI Package

Comprehensive AI integration library with OpenAI, Anthropic, and specialized AI tools for the Forge
platform.

## Overview

The AI package provides enterprise-grade AI functionality including chat interfaces, content
moderation, product classification, and pre-built AI tools. Built on Vercel AI SDK with enhanced
features for production use.

## Key Features

- **OpenAI Integration**: GPT-4 models with lazy-loaded clients and graceful fallbacks
- **Anthropic Claude API**: Content moderation, sentiment analysis, and entity extraction
- **Product Classification**: AI-powered e-commerce categorization with confidence scoring
- **Enhanced Chat Hook**: Retry logic, token tracking, and error handling
- **Pre-built Tools**: Calculator, web search, date/time, and JSON utilities
- **UI Components**: Ready-to-use chat interface components
- **Structured Generation**: Zod schema-validated AI responses
- **Streaming Support**: Real-time text and object generation

## Basic Usage

### Enhanced Chat Hook

```typescript
import { useAIChat } from '@repo/ai';

const { messages, input, handleSubmit, appendWithRetry, regenerate, clear, tokenUsage } = useAIChat(
  {
    api: '/api/chat',
    maxRetries: 3,
  }
);
```

### Streaming Text Generation

```typescript
import { streamCompletion, generateCompletion } from '@repo/ai';

// Streaming response
const stream = await streamCompletion('Write a story about...');

// Complete response
const text = await generateCompletion('Summarize this text...');
```

### Structured Data Generation

```typescript
import { generateStructuredData } from '@repo/ai';
import { z } from 'zod';

const schema = z.object({
  title: z.string(),
  summary: z.string(),
  tags: z.array(z.string()),
});

const result = await generateStructuredData('Analyze this product description...', schema);
```

## Anthropic Integration

### Content Moderation

```typescript
import { anthropic } from '@repo/ai/providers/anthropic';

const analysis = await anthropic.moderateContent(['User generated content to analyze...']);

// Returns: toxicity, spam, adult content, hate speech scores
```

### Sentiment Analysis

```typescript
const sentiment = await anthropic.analyzeSentiment(['Customer review text...']);

// Returns: positive, negative, neutral classification
```

### Entity Extraction

```typescript
const entities = await anthropic.extractEntities([
  'Text containing people, places, organizations...',
]);

// Returns: persons, organizations, locations, dates
```

## UI Components

### Chat Interface

```typescript
import { Thread, Message } from '@repo/ai';

function ChatInterface() {
  const { messages } = useAIChat();

  return (
    <Thread>
      {messages.map(message => (
        <Message key={message.id} message={message} />
      ))}
    </Thread>
  );
}
```

## Advanced Features

### Batch Processing with Rate Limiting

```typescript
// Anthropic batch processing
const results = await anthropic.batchModerateSafely(contentItems, {
  batchSize: 10,
  delayMs: 1000,
});

// Product classification batch
const classifications = await batchClassifyProducts(products, {
  batchSize: 5,
  includeVectorSimilarity: true,
});
```

### Model Configuration

```typescript
import { models } from '@repo/ai';

// Access configured models
const response = await models.chat.generateText({
  messages: [{ role: 'user', content: 'Hello!' }],
});

const embeddings = await models.embeddings.embedMany(['text1', 'text2']);
```

## Error Handling

The package includes comprehensive error handling:

- **Rate limit retries**: Automatic backoff for API limits
- **Graceful degradation**: Fallbacks when API keys are missing
- **Type-safe errors**: Structured error responses
- **Token tracking**: Monitor usage across requests
