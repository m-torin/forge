# Rate Limit Package

API rate limiting and request throttling utilities.

## Overview

Provides rate limiting capabilities for API endpoints to prevent abuse and ensure fair usage across
the platform.

## Key Features

- Request rate limiting
- Multiple rate limit strategies
- Redis-based storage
- API endpoint protection
- Custom rate limit rules

## Installation

```bash
pnpm add @repo/rate-limit
```

## Basic Usage

```typescript
import { rateLimit } from '@repo/rate-limit';

const limiter = rateLimit({
  windowMs: 60 * 1000, // 1 minute
  max: 100, // 100 requests per minute
});

export default limiter(async (req, res) => {
  // Your API handler
});
```

## Strategies

Supports multiple rate limiting strategies:

- Fixed window
- Sliding window
- Token bucket
- Per-user limits

## Storage

Uses Redis for distributed rate limiting across multiple application instances.

## Integration

Integrates seamlessly with Next.js API routes and middleware for automatic rate limit enforcement.
