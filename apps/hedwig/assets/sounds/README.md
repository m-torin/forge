# Sound Assets

Place audio files here for the AudioService:

## Required Files:
- `scan-success.mp3` - Played when barcode scan succeeds
- `scan-error.mp3` - Played when barcode scan fails
- `notification.mp3` - General notification sound
- `button-press.mp3` - UI feedback sound
- `alert.mp3` - Critical alert sound

## Recommended Specifications:
- Format: MP3 or M4A
- Duration: < 2 seconds
- Sample Rate: 44.1 kHz
- Bit Rate: 128 kbps
- File Size: < 100 KB each
